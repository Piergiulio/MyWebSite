The Darkness, Your Images




Welcome where everything starts traveller, press New Game to start a new fantastic quest!!!
Once you saved your data at least once you can use the "Load Menu" to start the game from where you left!
To view the Main Menu you will need to press Enter on your Keyboard and Esc to come out!
Use "Item" inside the Main Menu to see all the items aquired so far during your quest !
Use "Skill" inside the Main Menu to see all the skills each Charachter learned so far !
Use "Equip" in the Main Menu to see which Weapons/Armors your charachter is equipped with and change them !
Use "Status" inside the Main Menu to have information on the current statistic of each charachter of the party!
Each Character in the game got their own growing pattern and special techniques, you decide your favorites!
Each Skill in the Game got its unique parameters! Some are stronger against certain types of Monsters!
Each Monster in the game got their own styles and techniques, you need to find the best way to defeat them!
To become stronger you will need to Level Up! Defeat various enemies to get EXPerience points and Gold!
If you don't get enough experience and fight against enemies stronger than you the Game is Over for your party!
