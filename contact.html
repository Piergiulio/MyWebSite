FAQ's

Q1: Where I can download the file for The Darkness?
A1: You can find the link for downloading the file in our Media Gallery page.
Q2: I've downloaded the file how can I play ?
A2: Unzip the file, go inside The Darkness folder and install the file .exe.
Q3: The File is installed but it seems as the game won't start!
A3: Check which version you downloaded and if it's supported by your System, otherwise try to restart the computer after the installation.
Q4: How can I get a Screenshot while playing?
A4: The screenshot is taken by pressing F10.
Q5: I've used all the saving slots, how can I create a new saving slot?
A5:Go inside the folder "save" and create a new file with .sav extention!
Be careful you can only have four at the time, so the only first four files will be displayed inside the game, so you will have to move the new file on the top first places inside the folder.
Q6: Where I can find your Channel on Youtube?
A6: You can find us at the domain www.Youtube/TwincuzP.com.
Q7: I've sent my CV but never get an answer
A7: We scrutinize all the CV and carefully decide who meet the requirements. Allow us at least 28 days to get back to you! We will answer either if you pass or not.
Q8:My Cv got accepted , now what is going to happen?
A8: You don't have to do anything else, we will call you to decide a date and time that suit you for a job interview. Following the interview you will be given the date of your trial Shift. Best Of Luck!
